## use the command ```print(main(template, model, ascent, espace))``` in python shell
- template -> character list
- model -> character pictures
- ascent -> self-explanatory
- espace -> lengh of space character

## for instance, ```print(main("template.txt", "GHR M32.png", 28, 8))```

the glyphs are drawn on a 25x33 invisible grid
